# Lista de Exercícios - Semana 9: Manipulação de Strings

Esta lista aborda problemas envolvendo operações com strings em linguagem C.

## Exercícios Resolvidos
- Problema 1: Contagem de caracteres
- Problema 2: Verificação de palíndromos
- Problema 3: Substituição de palavras
- Problema 4: Formatação de texto

## Como executar
Compile os arquivos `.c` correspondentes aos exercícios usando o GCC:
```bash
gcc problema1.c -o problema1
./problema1
```
Repita para os demais problemas.
