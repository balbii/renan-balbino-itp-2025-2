# Lista de Exercícios - Semana 7: Vetores e Matrizes

Esta lista aborda problemas relacionados à manipulação de vetores e matrizes em linguagem C.

## Exercícios Resolvidos
- Problema 1: Manipulação de vetores
- Problema 2: Operações com matrizes
- Problema 3: Busca e ordenação
- Problema 4: Soma de elementos

## Como executar
Compile os arquivos `.c` correspondentes aos exercícios usando o GCC:
```bash
gcc problema1.c -o problema1
./problema1
```
Repita para os demais problemas.
